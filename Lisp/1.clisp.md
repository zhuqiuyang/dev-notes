```lisp
(+ 1 2) ; an s-expression
```

### 2.2 evaluate
表达式内的函数调用都会被evaluate
除非使用`quote`或者`'`, 用户保护表达式不被求值.

###2.3 Data
- lisp 有两种特殊的数据类型, symbol 和 list
```lisp
> 'Artichoke
ARTICHOKE     ; symbol
> '(my 3 "Sons")
(MY 3 "Sons") ; list
> (list 'my (+ 2 1) "Sons")
(MY 3 "Sons") ; list
```
- `()`and `nil` 都是空list

(+ 1 2) ; an s-expression

#### function 作为对象

`'` 作为 quote 的缩写，也可以用 `#'` 作为 `function` 的缩写:

```sh
> (function +)
#<Compiled-Function + 17BA4E>

> (apply #'+ '(1 2 3))
6
```

### 2.2 evaluate

表达式内的函数调用都会被 evaluate
除非使用`quote`或者`'`, 用户保护表达式不被求值.

###2.3 Data

- lisp 有两种特殊的数据类型, symbol 和 list

```lisp
> 'Artichoke
ARTICHOKE     ; symbol
> '(my 3 "Sons")
(MY 3 "Sons") ; list
> (list 'my (+ 2 1) "Sons")
(MY 3 "Sons") ; list
```

- `()`and `nil` 都是空 list

### 语言要素

> 其他都是, 以下四点的扩展 or 约束

#### 1. operator

- 函数: `定义, 调用`
- 运算符: `算术, 逻辑`

#### 2. operands

- 基本类型(`强类型`): `int, string, array, boolean`
- 复合类型: `struct, interface`

#### 3. 抽象

- OOP: `Object(data + method)` <= `Class`

#### 4. 结构

- 循环: `for, while`
- 条件: `if`
